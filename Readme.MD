# federicodecillia.github.io

Personal site of Federico De Cillia. Enterprise AI leader and hands-on builder.

Built with Astro + Tailwind v4, deployed to GitHub Pages via GitHub Actions.

## Local development
```
npm install
npm run dev      # local dev server
npm run build    # static build to dist/
npm run preview  # preview the build
```

## Deploy
Pushing to `master` triggers the GitHub Actions workflow (`.github/workflows/deploy.yml`), which builds with Astro and deploys to GitHub Pages. Set repo Settings → Pages → Source = "GitHub Actions" once.
